# Production Readiness Checklist

## Purpose

This checklist defines minimum readiness before production launch.

## Product

- core flows complete,
- error/empty/loading states complete,
- support contact visible,
- feature flags configured.

## Security

- auth tested,
- authorization tested,
- tenant isolation tested,
- secrets managed,
- file upload controls,
- AI Gateway controls,
- sandbox isolation.

## Privacy/Legal

- privacy notice,
- terms,
- child safety policy,
- data processing agreement where needed,
- subprocessor registry,
- AI data policy.

## Infrastructure

- production environment,
- database backups,
- Redis/queue health,
- object storage,
- monitoring,
- alerts,
- error tracking,
- deployment rollback.

## Operations

- support workflow,
- incident response,
- admin access controls,
- billing support,
- customer success ownership.

## Acceptance Criteria

- launch owner signs off,
- security owner signs off,
- product owner signs off,
- rollback plan exists,
- first-week monitoring plan exists.
