# Pilot Launch Checklist

## Purpose

This checklist defines what must be ready before launching a school/institution pilot.

## Pilot Definition

Minimum pilot:

- 1-3 institutions,
- 50-300 learners,
- 2-10 teachers,
- 6-8 weeks,
- Level 1 or focused AI tutor path.

## Product Readiness

Required:

- learner onboarding,
- teacher dashboard,
- class management,
- Level 1 lessons/projects,
- AI mentor guardrails,
- progress tracking,
- support channel.

## Technical Readiness

Required:

- staging tested,
- production environment ready,
- backups enabled,
- monitoring enabled,
- AI usage metering,
- error tracking,
- basic admin tools.

## Legal/Compliance Readiness

Required:

- pilot agreement,
- privacy notice,
- data processing agreement,
- consent approach,
- child safety policy,
- support terms.

## Training Readiness

Required:

- teacher onboarding guide,
- admin setup guide,
- learner quick-start guide,
- support contacts,
- pilot schedule.

## Measurement Plan

Track:

- activation,
- weekly active learners,
- lesson completion,
- project completion,
- AI helpfulness,
- teacher time saved,
- support issues,
- willingness to continue/pay.

## Exit Criteria

Pilot is successful if:

- learners complete meaningful projects,
- teachers use dashboard,
- AI is rated helpful/safe,
- institution sees value,
- support burden is manageable,
- pricing conversation advances.
