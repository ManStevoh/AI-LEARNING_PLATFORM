# Master Roadmap

## Roadmap Principle

The full platform is large enough to become a major company. It should be built in staged releases, with the web version first and future mobile, desktop, offline, robotics, and WhatsApp channels designed into the architecture without delaying the first usable product.

## Phase 0: Documentation And Research

Goal:

- create the engineering playbook,
- verify assumptions,
- define architecture,
- define curriculum,
- define security requirements,
- define business model,
- define first release scope.

Deliverables:

- product vision,
- market research,
- standards registry,
- curriculum framework,
- system requirements,
- architecture diagrams,
- ADRs,
- AI architecture,
- security baseline,
- UI/UX direction,
- subscription model,
- implementation plan.

## Phase 1: Web Foundation

Goal:

- launch the online web platform foundation.

Core technologies:

- Laravel,
- PostgreSQL,
- Redis,
- React,
- Inertia.js,
- TypeScript,
- Tailwind CSS,
- object storage,
- queue workers.

Core features:

- authentication,
- roles,
- institution accounts,
- teacher accounts,
- learner accounts,
- classes,
- basic learning paths,
- basic dashboard,
- subscription foundation,
- AI gateway foundation.

## Phase 2: Level 1 Block Coding

Goal:

- deliver beginner block coding with games and animations.

Core features:

- Blockly workspace,
- custom beginner blocks,
- stage,
- sprites,
- movement,
- events,
- loops,
- variables,
- project save/load,
- teacher assignment,
- AI block explainer,
- generated JavaScript preview.

## Phase 3: Level 2 Blocks To Code

Goal:

- help learners understand the relationship between blocks and text programming.

Core features:

- generated JavaScript,
- generated Python where feasible,
- concept explanations,
- code highlighting,
- AI code explanation,
- block-to-code lessons,
- assessments connecting visual blocks to syntax.

## Phase 4: Level 3 Hybrid Editing

Goal:

- transition learners from block programming to editable code.

Core features:

- editable code snippets,
- controlled code editing,
- synchronization rules,
- AI hints,
- beginner debugging,
- syntax explanations,
- projects that require both blocks and code.

## Phase 5: Level 4 Browser IDE

Goal:

- introduce professional text coding with AI mentoring.

Core features:

- Monaco Editor,
- file explorer,
- coding exercises,
- code execution sandbox,
- tests,
- AI debugging,
- AI code review,
- assignment submissions,
- grading assistance,
- teacher review tools.

## Phase 6: Level 5 Career Tracks

Goal:

- prepare learners for real software development careers.

Tracks:

- frontend development,
- backend development,
- full-stack Laravel + React,
- Next.js,
- Flutter mobile,
- Python,
- databases,
- APIs,
- cloud,
- DevOps,
- AI engineering,
- cybersecurity,
- robotics and IoT,
- system design,
- interview preparation.

Core features:

- real-world projects,
- portfolio builder,
- Git integration,
- deployment guidance,
- certificates,
- mock interviews,
- capstone projects,
- employer-ready profile.

## Phase 7: Mobile App

Goal:

- launch learner and teacher mobile apps.

Recommended technology:

- Flutter.

Capabilities:

- downloaded lessons,
- offline progress,
- quizzes,
- AI chat where online,
- push notifications,
- parent reports,
- lightweight coding practice,
- sync when online.

## Phase 8: Desktop App

Goal:

- provide stronger offline coding and classroom lab experiences.

Recommended technology:

- Tauri or Flutter Desktop.

Capabilities:

- offline lessons,
- local projects,
- local code editor,
- sync queue,
- classroom lab mode,
- possible local AI in future,
- hardware/robotics integrations.

## Phase 9: WhatsApp Learning

Goal:

- reach learners and parents through a familiar low-friction channel.

Capabilities:

- lesson reminders,
- quizzes,
- AI Q&A,
- parent progress summaries,
- teacher broadcasts,
- assignment reminders,
- voice notes in the future.

## Phase 10: Enterprise And Regional Scale

Goal:

- support large institutions, NGOs, governments, and multi-country expansion.

Capabilities:

- advanced tenant controls,
- custom curriculum packs,
- country-specific reporting,
- data residency options,
- integrations,
- enterprise support,
- dedicated deployments where needed.
