# Level 1 Block Coding Curriculum

## Purpose

Level 1 introduces learners to programming through visual blocks, games, animations, stories, creativity, and debugging.

Target learners:

- ages 8-10,
- older beginners,
- primary school learners,
- coding club beginners.

## Level Goal

By the end of Level 1, learners should be able to create simple interactive animations and games using block programming concepts.

## Core Skills

Learners should understand:

- sequence,
- events,
- motion,
- looks,
- sound,
- repeat loops,
- simple conditionals,
- basic variables,
- debugging,
- project presentation.

## Pedagogy

Use:

- constructionism,
- guided play,
- Use-Modify-Create,
- short challenges,
- debugging routines,
- reflection.

## Unit 1: Meet The Coding Studio

Skills:

- identify stage,
- identify sprite,
- drag blocks,
- connect blocks,
- run project.

Project:

- make a character say hello and move.

Assessment:

- learner can connect an event block and action block.

## Unit 2: Events And Actions

Skills:

- when green flag clicked,
- when key pressed,
- when sprite clicked,
- cause and effect.

Project:

- interactive greeting card.

Assessment:

- learner explains which event starts each action.

## Unit 3: Motion And Coordinates

Skills:

- move,
- turn,
- x/y position,
- direction,
- reset position.

Project:

- character walks across the screen.

Assessment:

- learner can position a sprite and move it intentionally.

## Unit 4: Looks And Storytelling

Skills:

- say,
- think,
- change costume,
- show/hide,
- backdrop changes.

Project:

- two-character story.

Assessment:

- learner uses timing and visual changes to tell a sequence.

## Unit 5: Sound And Interaction

Skills:

- play sound,
- start sound,
- combine sound with motion,
- user-triggered sound.

Project:

- musical character or soundboard.

Assessment:

- learner connects user action to sound output.

## Unit 6: Repeat Loops

Skills:

- repeat block,
- repeated movement,
- repeated animation,
- loop count.

Project:

- dancing character animation.

Assessment:

- learner explains why a loop reduces repeated blocks.

## Unit 7: Simple Conditions

Skills:

- if condition,
- touching color/sprite,
- key pressed,
- simple decision.

Project:

- collect-the-star mini game.

Assessment:

- learner uses a condition to trigger an action.

## Unit 8: Score And Variables

Skills:

- create score variable,
- change score,
- reset score,
- display score.

Project:

- click-to-score game.

Assessment:

- learner can update and reset a score.

## Unit 9: Debugging Blocks

Skills:

- identify disconnected blocks,
- identify wrong event,
- identify wrong loop count,
- test one change,
- explain fix.

Project:

- fix a broken game.

Assessment:

- learner explains what was wrong and how they fixed it.

## Unit 10: Final Project

Project options:

- maze game,
- catch game,
- animated story,
- quiz game,
- simple simulation.

Required features:

- at least one event,
- motion,
- looks,
- loop,
- condition,
- score or variable,
- sound or backdrop,
- debugging reflection.

Assessment rubric:

- functionality,
- creativity,
- use of concepts,
- debugging,
- explanation,
- presentation.

## AI Mentor Behavior

AI should:

- explain blocks in simple language,
- ask guiding questions,
- give hints before answers,
- identify disconnected blocks,
- suggest testing one change,
- use age-appropriate examples,
- encourage creativity.

AI should not:

- build the whole project for the learner,
- shame mistakes,
- use advanced jargon without explanation.

## Teacher Dashboard Requirements

Teacher should see:

- unit progress,
- project submissions,
- common mistakes,
- skills practiced,
- learners needing help,
- final project rubric scores.

## Kenya/CBC Alignment

Level 1 supports:

- creativity and imagination,
- critical thinking and problem solving,
- communication and collaboration,
- digital literacy,
- learning to learn,
- self-efficacy.

Local project examples:

- matatu route animation,
- farm watering schedule,
- market day story,
- football scoring game,
- wildlife conservation story.

## Completion Criteria

Learner completes Level 1 when:

- all core units are completed,
- final project submitted,
- learner explains at least three blocks used,
- teacher or system confirms basic mastery.
